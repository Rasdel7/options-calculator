# Options Strategy Calculator 📈

Quantitative finance tool for options
pricing, Greeks analysis and strategy
payoff visualization.

## Live Demo
[Click here](YOUR_STREAMLIT_URL)

## Features
- Black-Scholes call/put pricing
- All 5 Greeks: Delta, Gamma, Theta, Vega, Rho
- 12 strategy payoff diagrams
- Implied volatility calculator (Newton-Raphson)
- 3D volatility surface
- Theta decay visualization
- Greeks dashboard vs stock price
- Strategy comparison guide

## Strategies Covered
Long/Short Call/Put, Bull Call Spread,
Bear Put Spread, Bull Put Spread,
Long/Short Straddle, Long Strangle,
Covered Call, Protective Put

## Tools Used
- Python, Streamlit, SciPy, Plotly, NumPy

## How to Run Locally
pip install streamlit pandas numpy plotly scipy
streamlit run app.py
